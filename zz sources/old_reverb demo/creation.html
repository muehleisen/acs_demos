Sound Creation

Speech

To create the sounds used in this demo, a number of steps were involved.

First, the room designs were created and analyzed with the CATT acoustics program. CATT generated an impulse response for a given source and receiver location.

The impulse responses were generated using the generic head related transfer function (HRTF) as supplied with CATT and equalized for Sennheiser HD440 headphones.

Then, the impulse response was convolved with the anechoic samples, again using CATT acoustics, to generate the sounds in the different rooms. Finally, the sounds were adjusted so that the maximum amplitude was the same in all cases for better comparison using Cooledit96
